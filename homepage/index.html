Blobby Volley 3D is a 3D-version of the popular 2D-game Blobby Volley. The game was developed in the course of the seminar "Echtzeitcomputergrafik" (Real Time Rendering) at HPI and uses the Virtual Rendering System (VRS).
Our goal was to bring Blobby Volley to the third dimension while staying as close to the original as possible.

Features

  • Animated, semi-transparent, 3-dimensional blobbs
  • Realistic ball physics by integration of a physics engine (Newton Game Dynamics)
  • Several arenas (beach, heaven, city)
  • Classic rules or tie-break
  • Single-Player-Mode (against computer) and 1-on-1 (local)
  • Several camera perspectives
  • Controls: mouse or keyboard
  • Sound
  • dynamically sized shadows to help locate the ball

Controls

Blobb-movement: arrow keys|W, A, S, D|mouse
Jump: space|Q|left mouse button
Show menu: Esc
Pause: P
Change camera view: F2-F5
Camera height: F7
Camera distance: F8
Toggle extended camera animation: F9

Future

We intend to develop the game a little further. Planned additions are:
  • Multi player via network (that's the least thing we'd like to do)
  • 2-on-2 mode
  • more settings
  • graphical enhancements
Therefore we have also given public read access to our SVN repository (see Downloads section). There you'll find newer versions (or can contribute yourself).
